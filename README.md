# Typora-themes

## 介绍
Typora自用主题

## 提示

1.  version 0.11.8(bata)是最后的免费版本；

![2021-12-29_16-57-16](images/beta.png)

## 使用说明

1.  「文件」 --> 「偏好设置」 --> 「主题」 --> 「打开主题文件夹」；
2.  将下载的`cajun`文件夹和`cajun.css`文件拷贝到打开的主题文件夹下；
3.  「主题」 --> 选择「cajun」重启生效；
